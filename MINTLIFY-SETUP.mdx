## Prerequisites

- Node.js v20 LTS (downgrade from v25+ if needed)
- GitHub repository with documentation files
- Mintlify account and dashboard access

## Setup Steps

### 1. **Create Configuration File**
   - Create `docs.json` in the **root directory** (not a subfolder)
   - Include required fields: `name`, `theme`, `favicon`, `navigation`

### 2. **Use Correct Theme**
   - Valid themes: `mint`, `maple`, `palm`, `willow`, `linden`, `almond`, `aspen`, `sequoia`
   - Example: `"theme": "mint"`

### 3. **Configure Navigation**
   - Use **object format with tabs**, not array format
   - Structure:
     ```json
     "navigation": {
       "tabs": [
         {
           "tab": "Docs",
           "groups": [
             {
               "group": "Group Name",
               "pages": ["path/to/page"]
             }
           ]
         }
       ]
     }
     ```

### 4. **File Structure**
   - Keep all `.mdx` files in **root or subdirectories** (e.g., `api-reference/`, `sdk/`)
   - Do NOT create a `docs` folder
   - Files must have frontmatter with `title`:
     ```mdx
     ---
     title: "Page Title"
     description: "Optional description"
     ---
     ```

### 5. **Colors Configuration**
   - Only use: `primary`, `light`, `dark`
   - Remove `background` (not supported)
   - Example:
     ```json
     "colors": {
       "primary": "#10b981",
       "light": "#34d399",
       "dark": "#059669"
     }
     ```

   - Default URL: `https://<project-id>.mintlify.app`
   - Custom domain: Set up in dashboard settings
