# Movies API Documentation

This is the official documentation for the Movies API Platform built with Mintlify.

## Features

- 5 RESTful CRUD endpoints
- Interactive API playground
- Multi-language SDK support (JavaScript, C#)
- Comprehensive error documentation

## Getting Started

1. Install Mintlify CLI: `npm install -g mintlify`
2. Run development server: `mintlify dev`
3. Open http://localhost:3000

## Project Structure

```
├── api-reference/
│   └── endpoints/          # API endpoint documentation
├── sdk/                    # SDK documentation
├── support/               # Support documentation
├── introduction.md        # Homepage
└── mint.json             # Mintlify configuration
```

## Deployment

Deploy to Mintlify Cloud by connecting your Git repository at https://mintlify.com

## Customization

Edit `mint.json` to customize:

- Site name and colors
- Navigation structure
- API base URL
