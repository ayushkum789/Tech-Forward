# Quick Start Guide

Get up and running with the Movies API quickly.

## Prerequisites

- API access credentials
- Basic understanding of REST APIs
- HTTP client (curl, Postman, or programming language)

## Your First Request

Make your first API call to retrieve all movies:

```bash
curl https://tech-fwd-poc-production.up.railway.app/api/movies
```

## Next Steps

Explore the [API endpoints](/api-reference/endpoints/get-all-movies) to learn more about available operations.
